Setup Guide

Follow these simple steps to install and configure Sentinel MK2

This extension was developed for the HackHazards 25 hackathon and requires manual installation in developer mode.

1

Download the Extension

Download the unpacked extension from our Download Page or directly from the GitHub repository (.zip folder).

Download Extension
2

Enable Developer Mode

Open Chrome and navigate to chrome://extensions/ or go to Menu → More Tools → Extensions.

Toggle on the "Developer mode" switch in the top-right corner of the Extensions page.

3

Load Unpacked Extension

Click the "Load unpacked" button, then navigate to and select the folder where you downloaded the extension files.

Note: If you downloaded from GitHub, select the "dist" folder that contains the manifest.json file.
4

Add Groq API Key

Click the Sentinel MK2 icon in your browser toolbar, then go to the Settings tab (gear icon).

Enter your Groq API key in the designated field and save your settings.

Tip: Don't have a Groq API key yet? See the "API Configuration" tab for instructions.
5

And Voilà!

The Sentinel MK2 extension is now installed and ready to use. You should see the icon in your browser toolbar.

Click the icon to open the extension and start enjoying enhanced web security!

Sentinel MK2 includes screen content analysis to detect visual phishing attempts. To enable this feature, you'll need to grant screen capture permissions.

1

Enable Screen Capture

Click the Sentinel MK2 icon in your browser toolbar to open the extension popup.

Navigate to the "Screen Analysis" tab.

2

Grant Permissions

Click the "Enable Screen Analysis" button in the extension.

When prompted, select the screen or window you want to capture and click "Share".

3

Configure Settings

Adjust screen capture settings according to your preferences:

  • Capture resolution
  • Analysis frequency
  • Privacy zones (areas to exclude from analysis)
4

Start Protection

Screen content analysis will now run in the background, alerting you to potential security threats.

Privacy Note: All screen analysis happens locally on your device. No screenshots are sent to external servers.

To use the AI-powered security features of Sentinel MK2, you'll need to add a Groq API key. Follow these steps to obtain and configure your free API key.

1

Create a Groq Account

Visit the Groq Console and sign up for a free account.

Go to Groq Console
2

Generate an API Key

In the Groq Console, navigate to the API Keys section.

Click "Create API Key" and give your key a name (e.g., "Sentinel MK2").

3

Copy Your API Key

Copy the generated API key. Important: This key will only be shown once, so make sure to save it.

Security Note: Keep your API key private and never share it publicly.
4

Add the API Key to Sentinel

Click the Sentinel MK2 icon in your browser toolbar, then go to the Settings tab.

Paste your Groq API key in the designated field and click "Save Settings".

5

Verify Connection

To verify your API key is working correctly, go to the "AI Chat" tab in the extension and ask a simple security question.

If you receive a response, your API key is configured correctly!
