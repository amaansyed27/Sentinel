Setup Guide

Installing Sentinel MK2 Extension

This extension was developed for the HackHazards 25 hackathon and requires manual installation in developer mode.

1. Download the Extension

Download the unpacked extension from our Download Page or directly from the GitHub repository (dist folder).

2. Enable Developer Mode

Open Chrome and navigate to chrome://extensions/ or go to Menu → More Tools → Extensions.

Toggle on the "Developer mode" switch in the top-right corner of the Extensions page.

3. Load Unpacked Extension

Click the "Load unpacked" button, then navigate to and select the folder where you downloaded the extension files.

Note: If you downloaded from GitHub, select the "dist" folder that contains the manifest.json file.

4. Add Groq API Key

Click the Sentinel MK2 icon in your browser toolbar, then go to the Settings tab (gear icon).

Enter your Groq API key in the designated field and save your settings.

Tip: If you don't have a Groq API key yet, see the "API Configuration" tab for instructions.

5. And Voila!

The Sentinel MK2 extension is now installed and ready to use. You should see the icon in your browser toolbar.

Click the icon to open the extension and start enjoying enhanced web security!

Setting Up Screenpipe

Screenpipe is a companion application that enhances Sentinel MK2's ability to analyze screen content for security risks.

Important: Screenpipe is currently only available for Windows.

1. Open Terminal

Open Windows PowerShell or Command Prompt with normal user permissions (no need for administrator rights).

Tip: You can search for "PowerShell" in the Start menu to find it quickly.

2. Run Installation Command

Copy and paste the following command into your terminal and press Enter:

iwr get.screenpi.pe/cli.ps1 | iex

This command will download and install the Screenpipe application on your computer.

3. Launch Screenpipe

After installation completes, type the following command to launch Screenpipe:

screenpipe.exe

This will start the Screenpipe application, which will run in the background.

4. Screenpipe is Ready

Once Screenpipe is running, you'll see its icon in your system tray. The application is now ready to work with Sentinel MK2.

5. Usage Note

Always run Screenpipe before using the Sentinel MK2 extension for screen content analysis features.

Important: You'll need to launch Screenpipe each time you restart your computer or want to use the screen analysis features.

API Configuration

Sentinel MK2 uses the Groq API for AI-powered security analysis. Follow these steps to set up your API key.

Get a Groq API Key

Visit the Groq Console to create an account and obtain an API key.

  1. Create a new account or log in
  2. Navigate to the API Keys section
  3. Click "Create API Key"
  4. Give your key a name (e.g., "Sentinel MK2")
  5. Copy your new API key

Note: Groq offers free API access with limits that should be sufficient for hackathon demonstration purposes.

Configure Sentinel MK2

Add your API key to Sentinel MK2:

  1. Click the Sentinel MK2 icon in your browser toolbar
  2. Navigate to the "Settings" tab (gear icon)
  3. Find the "API Configuration" section
  4. Enter your Groq API key in the field provided
  5. Click "Save" to store your API key

Security tip: Your API key is stored securely in Chrome's extension storage and is not shared with any third parties.

Test the API Connection

To verify that Sentinel MK2 can connect to the Groq API:

  1. Go to the "AI Chat" tab
  2. Ask a security-related question like "Is this website secure?"
  3. If properly configured, you should receive an answer from the AI

Usage tip: Be specific with your questions to get the most accurate security insights.
