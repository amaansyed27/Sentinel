Behind the Scenes: Hackathon Journey

Tech Stack

  • Chrome Extension APIs for browser integration
  • Groq API with LLaMa 3 70B model
  • Screenpipe for screen content analysis
  • Modern vanilla JavaScript architecture
  • Webpack build system for optimization

Inspiration

Our project was born from the need to create more proactive security tools that can analyze not just website code but also visual content to protect users from sophisticated threats and scams.

We wanted to leverage AI to provide context-aware security insights that adapt to each user's browsing habits.

Innovations

  • ML-based risk assessment for link marking
  • Visual pattern recognition for detecting phishing
  • Context-aware AI security assistant
  • Real-time certificate analysis system
  • Adaptive security recommendations

Team Challenges

Our team overcame several challenges during the 36-hour hackathon:

  • Optimizing LLM prompts for security analysis
  • Implementing reliable screen capture integration
  • Building an accessible, user-friendly interface
  • Creating real-time link security indicators

Future Roadmap

  • Enhanced phishing detection algorithms
  • Cross-browser compatibility (Firefox, Safari)
  • Expanded threat intelligence database
  • Community contributed security rules
  • Performance optimizations

Hackathon Outcome

Our Sentinel MK2 project demonstrates the potential of AI-enhanced web security that provides actionable, context-aware protection for users of all technical backgrounds.

We're proud to showcase how browser extensions can be powerful tools for improving online safety in an increasingly complex threat landscape.
