Hackathon Implementation Notes

Technical Stack

  • Chrome Extension APIs
  • Groq API Integration
  • Screenpipe for screen content analysis
  • Vanilla JavaScript for core functionality
  • Webpack for build system

Inspiration

This project was inspired by the need for more proactive security tools that can analyze not just website code but actual visual content to protect users from sophisticated threats and scams.

Experimental Features

  • Link security marking using ML-based risk assessment
  • Screen content security analysis with visual pattern recognition
  • Context-aware AI assistant that understands security implications
