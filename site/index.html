About This Project

Sentinel MK2 was created during the HackHazards 25 hackathon organized by the Name Space community. This project showcases how browser extensions can be leveraged to enhance web security through AI integration.

As a hackathon project, Sentinel MK2 is provided as a developer extension that needs to be loaded unpacked. Follow our setup guide to get started.
