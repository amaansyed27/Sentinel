Required for screen analysis features

Screenpipe enhances Sentinel MK2 with screen content analysis capabilities.

This open-source tool enables real-time monitoring and analysis of screen content, helping detect visual phishing attempts and security risks.

Features:

  • 24/7 screen monitoring (local only)
  • Native OCR for text recognition
  • Privacy-focused design
  • Low resource usage (10% CPU, 4GB RAM)

Quick Installation Instructions

1

Download & Extract

Download the extension ZIP file and extract it to a folder on your computer.

2

Open Extensions Page

In Chrome, navigate to chrome://extensions/ or Menu → More Tools → Extensions.

3

Enable Developer Mode

Toggle on "Developer mode" in the top-right corner of the Extensions page.

4

Load Unpacked Extension

Click "Load unpacked" and select the extracted extension folder.

5

Configure API Key

Click the extension icon, go to Settings, and add your Groq API key.

Hackathon Project Note

This extension was created as a demonstration for the HackHazards 25 hackathon and is not available on the Chrome Web Store. As such, it must be loaded as an "unpacked extension" in developer mode.

The focus of this project was to showcase the integration of browser extensions with AI services (particularly Groq API) to enhance web security.
