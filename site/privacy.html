Privacy Policy

Last updated: April 2025

Introduction

Welcome to the Sentinel MK2 Privacy Policy. This document outlines how we collect, use, and protect your information when you use our browser extension.

Sentinel MK2 was created for the HackHazards 25 hackathon and is intended for educational purposes. We take privacy seriously and have designed the extension with user security and privacy in mind.

Information Collection

Sentinel MK2 is designed with privacy as a core principle. Here's what you should know about the data we collect:

Data Stored Locally

  • API Keys: Your Groq API key is stored locally in your browser's extension storage and is never transmitted to our servers.
  • Configuration Settings: Your preferences and settings for the extension are stored locally on your device.

Data Analyzed But Not Collected

  • Webpage Content: The extension analyzes web page content to provide security assessments, but this data is processed locally and is not collected or stored.
  • URLs: Website URLs are analyzed to check security, but are not logged or stored beyond the current browsing session.
  • Screen Content: When using the Screenpipe integration, screen content is analyzed locally on your device to detect security risks. This data does not leave your device.

External Services

Sentinel MK2 interacts with the following external services:

Groq API

When you use the AI-powered security features, queries are sent to the Groq API using your personal API key. These queries may include information about the current website you're visiting to provide context-relevant security analysis.

Please refer to Groq's Privacy Policy for information on how they handle data.

Screenpipe

The Screenpipe integration works locally on your computer. Screen content is analyzed locally and is not transmitted to external servers. For more information, please see Screenpipe's documentation.

Data Security

We implement several security measures to protect your information:

  • Data is processed locally on your device whenever possible
  • API keys are stored securely using browser extension storage mechanisms
  • No persistent logs are maintained of your browsing activity
  • Communications with external APIs use secure HTTPS connections

User Controls

Sentinel MK2 gives you control over your data:

  • You can disable specific features that you're not comfortable with
  • You can reset all settings to defaults at any time
  • Uninstalling the extension removes all stored data

Changes to This Policy

We may update our Privacy Policy from time to time. We will notify you of any changes by posting the new Privacy Policy on this page and updating the "Last Updated" date.

You are advised to review this Privacy Policy periodically for any changes. Changes to this Privacy Policy are effective when they are posted on this page.

Contact Us

If you have any questions about this Privacy Policy, please contact us:

  • By visiting our GitHub repository: https://github.com/amaansyed27/Sentinel
  • By creating an issue in the repository

This Privacy Policy was created for the HackHazards 25 hackathon project and is provided for educational purposes.
