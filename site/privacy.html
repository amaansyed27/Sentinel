Privacy Policy

HackHazards 25 Project

Introduction

Sentinel MK2 was created as a hackathon project for the HackHazards 25 event. We take your privacy seriously and have developed this policy to explain how we handle your data.

This privacy policy applies to the Sentinel MK2 browser extension and companion applications.

Data Collection

What data we collect

Sentinel MK2 may collect the following types of data:

  • URLs of websites you visit (to perform security analysis)
  • Website content for security analysis purposes
  • Screen content when the screen analysis feature is enabled via Screenpipe
  • API keys that you provide manually

How we use this data

The data collected is used exclusively for:

  • Performing real-time security assessments of websites
  • Analyzing links for potential security threats
  • Providing AI-powered security insights via the Groq API
  • Detecting potential phishing attempts or visual deception

Data Storage

Sentinel MK2 prioritizes your privacy in the following ways:

  • All analysis is performed locally within your browser whenever possible
  • Your Groq API key is stored securely in Chrome's extension storage system
  • Screen content is processed locally and is not persistently stored
  • No user data is transmitted to our servers, as this is a client-side only extension

Third-Party Services

Sentinel MK2 integrates with the following third-party services:

  • Groq API: When you use the AI chat feature, website content may be sent to Groq's API for analysis. Please review Groq's Privacy Policy for more information on how they handle data.
  • Screenpipe: If you enable screen content analysis, Screenpipe captures and processes screen content locally on your device. No screen content is transmitted outside your device unless explicitly initiated by you.

Your Rights

As a user of Sentinel MK2, you have the right to:

  • Disable any feature of the extension at any time
  • Uninstall the extension to stop all data processing
  • Clear your browser's extension storage to remove any stored preferences

Updates to this Policy

As this is a hackathon project, this privacy policy may be updated if the project is developed further. Any significant changes will be communicated through the project's GitHub repository.

Contact Information

If you have any questions about this privacy policy, please reach out through the GitHub issues page.

Hackathon Context

Please note that Sentinel MK2 was developed as part of the HackHazards 25 hackathon and is primarily a demonstration project. It was not designed for large-scale deployment or commercial use.
